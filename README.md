## 📈 Caso de Estudio Destacado: Backend API
Dentro de este portafolio, desarrollo un caso de estudio sobre una **API de Gestión de Ventas**. 

### Desafíos Superados:
1.  **Integridad de Datos:** Implementación de restricciones de llaves foráneas y transacciones.
2.  **Arquitectura MVC:** Separación de responsabilidades para un código mantenible.
3.  **Optimización:** Reducción de tiempos de respuesta mediante *Eager Loading* con Sequelize.
