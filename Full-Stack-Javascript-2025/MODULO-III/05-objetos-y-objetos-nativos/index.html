Checklist rápido (lo que debe quedar claro)

  • Un objeto agrupa propiedades (datos) y métodos (funciones).
  • Punto: obj.prop cuando la clave es simple. Corchetes: obj["prop"] cuando hay espacios o la clave viene en variable.
  • Podemos agregar, modificar y eliminar propiedades con delete.
  • Math ayuda con aleatorios y redondeos (random / floor / round / ceil).
  • String ayuda a limpiar texto: trim, toLowerCase, replace, normalize.
  • match + expresiones regulares permite extraer números desde texto.
