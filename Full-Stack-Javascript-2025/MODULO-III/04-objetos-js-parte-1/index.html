Checklist rápido (docente)

  • Objeto = contenedor con propiedades (datos) y métodos (acciones).
  • Acceso con . cuando el nombre es simple; con [] cuando es dinámico o tiene espacios.
  • Puedo agregar/modificar propiedades después de crear el objeto; para eliminar uso delete.
  • Math es global: no se instancia; sirve para cálculos y números aleatorios.
  • String ofrece métodos para buscar y transformar texto (includes, replace, trim, etc.).
