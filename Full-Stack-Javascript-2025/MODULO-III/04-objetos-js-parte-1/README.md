# Conceptos básicos de Objetos en JavaScript — Parte I (AE3.5)

Incluye:
- `index.html` con panel de demos (Objetos, Math, String) y salida en pantalla.
- `assets/js/app.demo.js` con demos hiper comentadas.
- `assets/js/ejercicios/` con ejercicios alumno + solución.

## Cómo ejecutar
1) Abre `index.html` con Live Server (VS Code) o doble click.
2) Usa los botones del panel para ejecutar demos.
3) Mira la salida en pantalla y/o consola (F12).
