Abre la consola del navegador para ver los resultados. Atajo: F12 o Ctrl/Cmd + Shift + I → Console.

¿Qué archivo JS está cargado?

Por defecto está cargado assets/js/app.demo.js (demos para la clase). Para trabajar ejercicios, cambia el src del script al archivo del ejercicio. 

<script src="assets/js/app.demo.js"></script>

Ejercicios

  1. Contador de visitas
  2. Simulador de temperatura
  3. Calculador de edad

Los ejercicios están en assets/js/ejercicios/ y vienen en versión .alumno.js (con TODO) y .solucion.js.
