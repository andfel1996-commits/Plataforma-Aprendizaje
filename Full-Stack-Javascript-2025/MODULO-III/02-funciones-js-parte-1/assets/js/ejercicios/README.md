# Ejercicios — Funciones en JavaScript (Parte I)

## Cómo ejecutar un ejercicio
1. Abre `index.html`.
2. Abre la consola del navegador (F12 → Console).
3. En `index.html`, comenta el script de `app.demo.js`.
4. Descomenta el script del ejercicio que quieras.

> Cada ejercicio tiene 2 versiones:
> - `*.alumno.js` (con TODO)
> - `*.solucion.js` (resuelto)
