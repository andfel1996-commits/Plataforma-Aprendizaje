# Ejercicios — Funciones JS Parte II ✅

## Cómo usar
1. Abre el archivo **.alumno.js** del ejercicio.
2. Lee los comentarios (están pensados como “guión”).
3. Completa el código donde dice **TODO**.
4. Prueba con los casos de prueba del final.
5. Si te atascas, revisa el archivo **.solucion.js**.

---

## Ejercicio 01 — Calculadora de Sueldos
Crea:
`calcularSueldoNeto(nombreEmpleado, sueldoBase, ...bonificaciones)`
- Suma base + bonos, descuenta 13%, retorna neto.

## Ejercicio 02 — Registro de Asistencia
Crea:
`registrarAsistencia(materia, totalClases, ...asistentes)`
- Lista asistentes, calcula % asistencia, retorna %.

## Ejercicio 03 — Total de Compra + Callback
Crea:
- `calcularTotalCompra(nombreCliente, ...precios)`
- `aplicarOperacion(total, callback)`
Callbacks:
- `aplicarDescuento10`
- `aplicarIVA19`
