Checklist rápido (docente)

          
    
            
  • Si falta un parámetro → queda undefined.
    • 
            
  • Si opero con undefined en matemática → aparece NaN.
    • 
            
  • Si no hay return → la función devuelve undefined.
    • 
            
  • Variables locales viven dentro de la función; globales viven afuera (¡cuidado!).
    • 
            
  • ...rest captura muchos argumentos como array; debe ir al final.
    • 
            
  • Callback = pasar una función como parámetro para personalizar comportamiento.
