# Cómo ser un Fullstack JavaScript y no morir en el intento

Material complementario

## Descripción

Este repositorio reúne material de apoyo para quienes están dando sus primeros pasos en el camino **Fullstack JavaScript**.  
La idea es acompañarte módulo a módulo, explicando los conceptos fundamentales con ejemplos simples, ejercicios guiados y recursos que puedas revisar con calma después de cada clase.

No necesitas experiencia previa en programación: el contenido está pensado para ir **paso a paso**, desde la lógica básica y el pseudocódigo, hasta el uso de JavaScript en el frontend y backend, integrando herramientas modernas y buenas prácticas de la industria.
