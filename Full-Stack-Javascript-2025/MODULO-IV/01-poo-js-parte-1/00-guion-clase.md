# Guión de clase — POO en JavaScript (Parte I) — AE1.1

## Objetivo
Comprender **clase vs instancia**, y aplicar pilares POO:
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción

## Secuencia sugerida (60–90 min)
1) Activación: objetos del mundo real (usuario, vehículo, cuenta).
2) Demo 1: Objeto literal (rápido, no plantilla).
3) Demo 2: Constructora + prototype (métodos compartidos).
4) Demo 3: class + herencia (extends/super) + override.
5) Demo 4: Vehículos (polimorfismo claro).
6) Demo 5: Encapsulamiento con #privado + métodos.
7) Demo completa: función de negocio que oculta detalles (abstracción).
8) Trabajo alumnos: 3 ejercicios con solución.

## Cierre
Repasar:
- 3 formas de crear objetos
- 4 pilares (con un ejemplo)
