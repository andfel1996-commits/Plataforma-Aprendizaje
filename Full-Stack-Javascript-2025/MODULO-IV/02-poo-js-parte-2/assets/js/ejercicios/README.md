# Ejercicios — POO II (prototype + JSON)

## Cómo ejecutar (rápido)
Tienes 2 opciones:

### Opción A) Copiar/pegar en consola
1) Abre `index.html`
2) Abre consola (F12)
3) Copia el contenido del ejercicio y pégalo en la consola

### Opción B) Usar `demo.html`
1) Abre `demo.html`
2) Descomenta SOLO 1 `<script>` a la vez (el ejercicio que quieras)
3) Abre consola (F12) para ver la salida
