# POO en JavaScript (Parte II) — AE1.2 — Pack Docente

Este pack está alineado a la clase **Orientación a objetos en JavaScript — Parte II**.

Incluye:
- `00-guion-clase.md` (paso a paso + “qué decir”)
- `index.html` (demo interactiva con botones)
- `demo.html` (demo rápida sin UI para ejecutar ejercicios por script)
- `assets/js/app.demo.js` (demo docente hiper comentada)
- Ejercicios (alumno + solución):
  - `01-empleados-prototype` (función constructora + prototype + JSON)
  - `02-productos-json-objetos` (objetos literales + método + JSON)

## Uso rápido
1) Abre `index.html` con Live Server o en el navegador.
2) Abre consola (F12).
3) Ejecuta los botones en orden sugerido.
4) Para ejercicios, revisa: `assets/js/ejercicios/README.md`
