# Ejercicios — ECMA6 (ES6+) Clase II

Cada ejercicio corresponde a una sección de la demo.

## Cómo trabajar
1) Abre el archivo del ejercicio.
2) Lee el enunciado (comentarios).
3) Completa los TODO.
4) Ejecuta en consola o en el navegador.

> Para el navegador: usa Live Server con `exercise-runner.html`.
