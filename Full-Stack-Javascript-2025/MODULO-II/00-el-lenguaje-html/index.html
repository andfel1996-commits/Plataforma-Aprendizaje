¿Qué es HTML?

HTML es un lenguaje de marcado (no es un lenguaje de programación) que nos permite estructurar el contenido: títulos, párrafos, listas, enlaces, etc.

Documentación oficial recomendada: MDN Web Docs: HTML

Si quieres ver el bloque de accesibilidad, haz clic aquí: Accesibilidad

Estructura base del documento

  1. <html>: contenedor raíz del documento (todo va dentro).
  2. <head>: configuración y metadatos (no visible).
  3. <body>: contenido visible.

Regla mental simple: HEAD = configuración / BODY = contenido.

HTML5 semántico

¿Por qué usar semántica?

Porque mejora: accesibilidad (lectores de pantalla), SEO (estructura clara), y mantenibilidad (código más ordenado).

Ejemplo: Un <header> describe un encabezado, un <nav> un menú de enlaces, y <footer> el pie del sitio.

Accesibilidad (ideas básicas)

Accesibilidad significa que la web pueda ser usada por personas con distintas capacidades y también en distintos contextos (teclado, lector de pantalla, etc.).

  • Usar títulos en orden (h1, h2, h3...)
  • Usar etiquetas semánticas: main, nav, header
  • Agregar aria-label cuando algo necesita “nombre” extra (como un nav)
