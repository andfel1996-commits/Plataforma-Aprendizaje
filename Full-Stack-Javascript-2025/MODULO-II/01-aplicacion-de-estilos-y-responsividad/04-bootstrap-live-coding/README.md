# Bootstrap — Live Coding (Parte I)

Este paquete contiene **3 mini-proyectos** (uno de demo y dos ejercicios) para la clase:

1) **01-demo-bootstrap-parte-1**
   - Objetivo: ver el impacto de Bootstrap al instante (CDN + navbar + grillas + tabla + formulario + alerta).

2) **02-ejercicio-1-wallet**
   - Objetivo: practicar **grillas + tablas** con un caso de “billetera digital”.

3) **03-ejercicio-2-registro**
   - Objetivo: practicar **componentes** (navbar + formulario + alerta) con un caso de registro.

## Cómo ejecutar
- Abre cada carpeta y ejecuta el archivo `index.html` en el navegador.
- Recomendado: usar **Live Server** en VS Code (clic derecho → “Open with Live Server”).

## Reglas simples
- Bootstrap se integra con:
  - `<link ...bootstrap.min.css>` en el `<head>`
  - `<script ...bootstrap.bundle.min.js>` antes de cerrar `</body>`
- Tu archivo CSS propio se carga **DESPUÉS** del CSS de Bootstrap para poder sobrescribir estilos.
