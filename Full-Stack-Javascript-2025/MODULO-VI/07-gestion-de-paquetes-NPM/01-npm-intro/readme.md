## Objetivo practicar
### Paso 0
`npm init -y` (Iniciamos el proyecto para poder trabajar con dependencias)

- `npm install` ( Instalar dependencias )
- `package.json` : `dependencias` , `devDependencias`, `scripts (npm run <comando>)`
-  Diferencias entre dependencias de producción y de desarrollo
- `npm update` / `npm uninstall`

## Pasos
```bash
npm install
npm run start
npm run dev
npm run info
```

## 1) Actualizar dependencias
```bash
npm update
```

## 2) Eliminar una dependencia
```bash
npm uninstall dayjs;
```

## 3) Instalar dependencia por version
npm install `nombre_del_paquete>`@`numero_version`
