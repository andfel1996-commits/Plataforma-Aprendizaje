# 01 - Levantar una aplicación con Node.js (`node app.js`)

## Objetivo
Entender:
- Qué hace Node al ejecutar un archivo.
- Por qué a veces el proceso termina y a veces queda corriendo.

## Ejecutar
```bash
npm install
npm start
```

## Actividad
1) Ejecuta con `mantenerVivo = true` (queda corriendo).
2) Cambia a `false` y ejecuta de nuevo (termina solo).
