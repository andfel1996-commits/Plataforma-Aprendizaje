Caso de Estudio: API de Gestión Backend

Descripción de la actividad

Desarrollo de una API REST robusta para centralizar la gestión de inventario y pedidos, utilizando Node.js y Sequelize.

Desafío Principal

Implementar la integridad referencial en una base de datos compleja para evitar datos inconsistentes en transacciones financieras.

Solución Propuesta

Uso de transacciones SQL y lógica asíncrona mediante el patrón MVC, asegurando que cada operación de escritura sea segura.

Herramientas Técnicas

  • Node.js
  • Express
  • Sequelize
  • PostgreSQL
  • Postman

Métricas de Impacto

  • 100% de integridad de datos alcanzada.
  • Reducción del 25% en tiempos de respuesta de consultas.

Aprendizajes y Justificación

Reforicé el manejo de modelos asíncronos. Elegí este proyecto porque demuestra mi capacidad para resolver problemas de lógica de servidor real.
